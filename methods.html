Collection, Model & Views Methods

Added some helpers to dry out the code.

Collection.setup(n) - Creates a puzzle grid of n x n blocks. Each block contains 9 cells.
Collection.setTo(x, y, value) - Create a puzzle grid cell using x, y.
Collection.getBlock(n) - Accepts a Block number and returns all cells.
Collection.getLastCell(n) - Gets the last well of a grid.
Collection.getRow(n) - Gets all the cells in a grid row.
Collection.getColumn(n) - Gets all the cells in a grid column.

Model.getRow() - Returns the row number of the model cell.
Model.getColumn() - Returns the row number of the model cell.
Model.getIndex() - Returns the index of the bock in the grid.
Model.getBlock() - Returns the block number of the current cell.

View.CellView - Build an individual cell view.
View.BlockView - Build an individual block view.
View.PuzzleView - Build an puzzle view.
